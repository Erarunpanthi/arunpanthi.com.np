Engineering Drawing

1. Introduction

Drawing is the act or art of creating a picture, diagram, or representation using lines, typically with a pencil, pen, or crayon. Drawing represents objects on a two-dimensional sheet. These objects can be real or imagined by engineers, technicians, or designers.

Drawings are classified into two main types:

  • Artistic Drawing: Created by artists based on imagination, without strict rules. Different people may interpret these drawings differently.
  • Engineering Drawing: Created by engineers or technicians following standard rules, providing clear and consistent information about objects. It’s considered a universal graphic language.

1.1 Types of Engineering Drawing

  • Freehand Sketch: Drawn without instruments.
  • Instrumental Drawing: Drawn using instruments for accuracy.
  • Computer-Aided Drawing (CAD): Created using software like AutoCAD.

1.3 Basic Steps for Drawing

  • Understanding the Subject: Familiarize yourself with the object to be drawn.
  • Preparing the Drawing Surface: Ensure the drawing board is clean and the paper is secured.
  • Sketching the Outline: Use light pencil strokes to create the basic shape.
  • Adding Details: Gradually refine the drawing by adding intricate details.
  • Checking Proportions: Review and adjust proportions as needed.
  • Finalizing the Drawing: Ink the drawing and erase any pencil marks.

1.4 Techniques for Drawing

  • Line Quality: Vary line thickness to create depth and emphasis.
  • Shading: Use hatching and cross-hatching to represent light and shadow.
  • Perspective: Apply one-point or two-point perspective techniques for realistic depth.

1.5 Finalizing the Drawing

  • Inking: Once satisfied with the pencil drawing, carefully ink over the lines for permanence.
  • Erasing Guidelines: After inking, gently erase any pencil guidelines without damaging the inked lines.
  • Presentation: Consider mounting the drawing on a presentation board for display.

1.8 Scales

Scales help represent objects proportionally on a drawing sheet. Common types:

  • Full-Size Scale (1:1): The drawing size is the same as the actual object.
  • Reducing Scale: The drawing is smaller than the actual object, e.g., 1:2, 1:5, 1:10.
  • Enlarging Scale: The drawing is larger than the actual object, e.g., 2:1, 5:1, 10:1.

1.9 Dimensioning

Dimensioning conveys the exact size, shape, and location of parts. Types include:

  • Dimension Lines: Indicate the size of an object, with arrowheads at each end.
  • Extension Lines: Extend from the object to show the exact points being measured.
  • Leader Lines: Point to specific features, providing additional information.
